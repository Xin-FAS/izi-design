## Demo

这是一个Demo组件，用来做文档示例

### 简单使用

使用线上版本测试

```jsx
import { Demo } from '@izid'

export default () => {
    return <Demo />
}
```

使用本地版本测试

```jsx
import { Demo } from 'izid'

export default () => {
    return <Demo />
}
```
