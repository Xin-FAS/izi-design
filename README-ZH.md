[English](https://github.com/Xin-FAS/izi-design/blob/main/README.md) | 中文文档
## ✨ 简介

基于 Ant Design 组件的二次封装，简化了组件的使用方式，适用于复杂的业务场景
- 封装一些业务组件
- 长期维护 (欢迎 PR)

## 📦 快速上手
推荐使用 `pnpm` 上手
```bash
pnpm add izid
```
